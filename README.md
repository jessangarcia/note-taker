
# Note Taker

This application is able to write, save, and delete notes for any task

## Installation

Install my-project with npm

```bash
  npm i
  npm install express
  npm install uuid
  heroku create
```
    
## Running Tests

To run tests, run the following command

```bash
  npm start
```

Used https://insomnia.rest/ to test the GET, POST, DELETE in localhost
